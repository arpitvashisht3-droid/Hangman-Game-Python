# 🎮 Hangman Game

A terminal-based Hangman game built in Python.

## Features

- 🎨 Colored terminal interface
- ❤️ Lives counter
- 🎲 Random word selection
- ✍️ Tracks guessed letters
- 🏆 Win/Lose messages
- ☠️ ASCII Hangman

## Technologies

- Python
- Colorama

## Run

```bash
pip install colorama
python3 main.py
